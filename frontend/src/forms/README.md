# Multi-Step Student Inquiry Form (/src/forms)

## Modules
- Step1/: Basic Info module (Full Name, Email, Mobile Number, City).
- Step2/: Course Info module (Qualification, Current Class, Interested Course, Interested Subject, Preferred Batch, Learning Mode).
- Step3/: Additional Info module (Queries, Career Goal, Remarks).
- Review/: Renders all fields aggregated for student review before final submit.
- Success/: High-aesthetic thank-you panel shown post-successful MongoDB insert.
