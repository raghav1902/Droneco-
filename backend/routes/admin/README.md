# Admin Routes (/routes/admin)

## Endpoints
- GET /api/admin/leads - Restricted: Admin (Full Lead Info)
- GET /api/admin/leads/:id - Restricted: Admin (Full details of a lead)
- POST /api/admin/users - Restricted: Admin (Create Receptionist)
- GET /api/admin/users - Restricted: Admin (List Users)
- DELETE /api/admin/users/:id - Restricted: Admin (Delete Receptionist)
- GET /api/admin/analytics - Restricted: Admin (Lead Reports)
