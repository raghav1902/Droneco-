# Admin Controller (/controllers/Admin)

## Purpose
Provides full control over leads, reports, receptionist accounts, and system operations.

## Actions (Placeholder)
- getAllLeads(req, res): Returns all leads with all fields visible.
- getLeadDetails(req, res): Returns full 3-step lead details by ID.
- manageUsers(req, res): CRUD operations on receptionist accounts.
- getAnalytics(req, res): Lead conversion dashboards and batch distribution reports.
